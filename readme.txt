Scanne votre site pour lister les images potentiellement non utilisées (posts, pages, produits WooCommerce, Elementor, ACF, options) + références dans les fichiers du thème actif. Export CSV + corbeille sécurisée.

= Utilisation =
- Allez dans **Médias → Unused Media Scanner**.
- Cliquez **Démarrer le scan** (scan par lots).
- Exportez le **CSV** ou envoyez à la **corbeille** (réversible).

= Sécurité & limites =
- Faites une sauvegarde (BD + /uploads) et, si possible, utilisez un environnement de staging.
- Les images injectées uniquement via JS ou shortcodes customs peuvent échapper à la détection.
- Analyse les fichiers du thème actif (enfant + parent) PHP/CSS/JS/HTML.
